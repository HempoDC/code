Vägen till IT-infrastrukturspecialist

Branschen kräver en djup bredd. Du behöver förstå hur datatrafik flödar (Nätverk), var tjänster körs (OS & Virtualisering), hur resurser skalas upp (Molnet), hur processer automatiseras (DevOps/IaC) och hur allt detta skyddas mot yttre hot (Säkerhet). Denna portal samlar de absolut viktigaste koncepten i en fokuserad studieplan.

Utbildning: IT-infrastrukturspecialist

Denna portal är utformad som ett komplement till yrkeshögskoleutbildningar eller självstudier som syftar till roller som IT-infrastrukturspecialist, Driftstekniker, Nätverkstekniker eller Systemadministratör.

Viktiga kompetensområden för rollen:

01

Nätverksdesign

Skapa stabila, redundanta lokala och globala nätverk, konfigurera brandväggar, routrar och switchar.

02

Server & Virtualisering

Driftsätta och underhålla Windows- och Linux-miljöer samt administrera hypervisorer som VMware och Hyper-V.

03

Molnarkitektur

Skala ut och underhålla resurser i molnmiljöer med fokus på Microsoft Azure (IaaS, PaaS, SaaS).

04

Automation (IaC)

Ersätta manuella installationsrutiner med kod. Konfigurera servrar med Ansible och provisera miljöer med Terraform.

Laborationsmiljöer

Simulator

Cisco Packet Tracer

Det viktigaste verktyget för att simulera switchar, routrar och nätverkskopplingar under dina nätverksstudier.

Hämta via NetAcad (Gratis efter konto-registrering)
Sandbox

Oracle VirtualBox

Skapa virtuella maskiner lokalt på din egen PC för att öva installation av Windows Server, Ubuntu och brandväggar.

Hämta via VirtualBox.org (Öppen källkod)
Molnövning

Azure Free Tier

Skapa ett gratis Azure-konto för att få tillgång till virtuella maskiner, virtuella nätverk (VNets) och App Services under 12 månader.

Registrera via Azure Portal (Kräver MS-konto)
