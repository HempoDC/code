Analyst Strategy Guidelines

  • Establish standard time-alignment baseline across target clocks immediately.
  • Check log source statuses to confirm Sysmon, Linux auditd, and HTTP logs are feeding the SIEM.
  • Create query outlines for rapid process tracking, logon audits, and AD ticket dumps.
  • Document exact timestamps (UTC) of actions to construct a chronological incident path.
  • Identify administrative parent process chains (like wmiprvse.exe or psexec.exe child commands).

UTC Time Alignment (Linux/Windows)

w32tm /tz; date -u
Check timezone offset settings on target systems to correlate timestamps accurately with the SIEM logs.

Tracing Executable Hash (PowerShell)

Get-FileHash -Algorithm SHA256 C:\path\to\suspect.exe
Compute executable hash on Windows target to search VirusTotal or look for known IOCs.

auditd Log Tracing (Linux)

ausearch -i -m SYSCALL -ts raw
Query raw auditd syscall events from local cache, translating numeric values to readable strings (-i).
